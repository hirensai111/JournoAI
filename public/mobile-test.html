Testing Guide

How to Test

  1. Open browser DevTools (F12)
  2. Enable device toolbar (Ctrl+Shift+M or Cmd+Shift+M)
  3. Test at these widths:
    • 320px - iPhone SE (small phones)
    • 375px - iPhone 12/13 (standard phones)
    • 390px - iPhone 14 Pro
    • 414px - iPhone 14 Plus
    • 768px - iPad (tablets)
    • 1024px - iPad Pro (large tablets)
  4. Verify no horizontal scrolling
  5. Check that all text is readable
  6. Confirm buttons have adequate touch targets (44px minimum)

✓ Success Criteria

  • No horizontal scrolling at any width
  • All text fits within containers
  • Font sizes are readable (minimum 14px)
  • Touch targets are at least 44x44px
  • Proper text wrapping for long words/URLs
  • Cards and containers adapt to screen width
